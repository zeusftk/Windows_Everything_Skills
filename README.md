# Windows Everything Skills

IPC 自动化技能集，为 AI Agent 提供 Windows 系统控制能力。

## 功能

### Desktop - 桌面自动化

| 功能组 | 说明 |
|--------|------|
| **鼠标操作** | 点击、移动、拖拽、滚动 |
| **键盘操作** | 文本输入、按键按下、快捷键组合 |
| **屏幕操作** | 全屏截图、窗口截图、获取屏幕尺寸、获取鼠标位置 |
| **窗口操作** | 查找窗口、列出所有窗口、查找UI元素、启动/关闭应用 |
| **剪贴板** | 读取/设置剪贴板内容 |
| **文件系统** | 读取、写入、复制、移动、删除文件，搜索文件，获取文件信息 |
| **PowerShell** | 执行 PowerShell 命令 |
| **虚拟桌面** | 创建、删除、切换虚拟桌面，移动窗口到指定桌面 |
| **多显示器** | 获取显示器信息、虚拟屏幕边界、DPI缩放比例 |
| **高级操作** | 检测对话框、等待延时、连接测试 |

### Web - 浏览器控制

| 功能组 | 说明 |
|--------|------|
| **浏览器控制** | 启动、停止、重启浏览器，获取浏览器状态 |
| **页面导航** | 导航到URL，获取当前URL和页面标题 |
| **页面交互** | 点击元素、填充输入框、滚动页面、按键操作、悬停 |
| **页面信息** | 截图、获取HTML内容、提取纯净文本 |
| **Cookie** | 获取页面Cookies |
| **页面结构** | 分析DOM结构、提取链接、查找可交互元素、查找按钮 |
| **等待操作** | 等待元素出现、等待文本内容稳定 |
| **JavaScript** | 执行JavaScript代码 |
| **标签页管理** | 获取、新建、关闭、切换标签页 |
| **任务管理** | AI任务执行、异步任务提交/查询/取消/获取结果 |
| **模拟人类** | 人类滚动、人类输入、模拟浏览、随机延迟 |

### WeChat - 微信集成

| 功能组 | 说明 |
|--------|------|
| **连接管理** | 连接微信、断开连接、获取状态、获取用户资料 |
| **会话管理** | 获取会话列表、获取所有未读消息统计 |
| **消息操作** | 发送文本/文件、获取聊天记录、获取未读消息 |
| **群组操作** | 获取群成员列表 |

### OCR - 文字识别

| 功能 | 说明 |
|------|------|
| 文字识别 | 对图像进行OCR文字识别 |
| OCR点击 | 通过OCR识别屏幕文本并点击指定位置 |
| OCR截图 | 截取屏幕并进行OCR识别 |

### OpenCLI - CLI适配器

| 功能组 | 说明 |
|--------|------|
| **适配器管理** | 列出适配器、获取适配器信息 |
| **执行/配置** | 执行CLI命令、验证YAML配置、获取统计和日志 |

### Embedding - 向量嵌入

| 功能 | 说明 |
|------|------|
| 健康检查 | 检查Embedding服务状态 |
| 生成向量 | 将文本转换为向量表示 |
| 相似度计算 | 计算文本间的语义相似度 |

### Web Info - 信息获取

| 功能 | 说明 |
|------|------|
| 信息搜索 | 通过搜索引擎获取相关信息 |
| 信息抓取 | 从URL获取网页内容并解析为结构化数据 |

### App Registry - 应用注册

| 功能 | 说明 |
|------|------|
| 列出应用 | 获取所有已注册应用列表 |
| 白名单管理 | 添加、获取、移除应用白名单 |

## 安装流程

1. 运行 FTK_Claw_Bot
2. 将 `create_ipc_skills` 添加到支持 skill 的 agent 或 coding 应用的 skills 文件夹下，如 OpenClaw、OpenCode、Hermes、Trea 等
3. 输入 "使用 create_ipc_skills 安装 ipc 技能"
4. 等待完成

## 使用方法

### Agent 对话方式

```
使用 create_ipc_skills 安装 ipc 技能
```

安装后即可通过自然语言调用 IPC API：

- "截图"
- "打开 Chrome 浏览器访问百度"
- "发送微信消息给张三说你好"
- "识别这张图片里的文字"
- "创建新的虚拟桌面"
- "搜索一下今天天气"

### Python API

```python
from clawbot.skills.create_ipc_skills.scripts.ipc_client import IPCClient

with IPCClient() as client:
    response = client.send('desktop_mouse_click', {'x': 100, 'y': 200})
```
